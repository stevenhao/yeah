To use gobot_client, visit and download the following link: https://www.dropbox.com/s/fby8xcr5cdbopc8/function.save.zip?dl=0. 

Unzip the file into this folder, GoBot/, making sure the file name is function.save. Start the GoBot client in the same way as a regular client:
python gobot_client.py [PORT] [IP ADDRESS]
